# JavaScript to TypeScript 실습

이 프로젝트는 `main` 브랜치에 있는 JavaScript 코드를 클론한 뒤, `TSver` 브랜치에서 TypeScript 코드로 변환해보는 실습 프로젝트입니다.

## 실습 목적

- JavaScript 코드와 TypeScript 코드의 차이 이해
- TypeScript의 타입 시스템 사용
- Interface, Union Type, Generic 개념 실습
- DOM 요소에 타입 지정하기
- 컴파일 과정을 통해 TypeScript가 JavaScript로 변환되는 흐름 이해

## 브랜치 구조

```text
main   → JavaScript 원본 코드
TSver  → TypeScript 변환 코드
```

## 실습 진행 방법

### 1. 저장소 클론

```bash
git clone 레포지토리주소
```

### 2. 프로젝트 폴더로 이동

```bash
cd 레포지토리이름
```

### 3. main 브랜치 코드 확인

```bash
git checkout main
```

`main` 브랜치에서는 JavaScript 버전의 코드를 확인할 수 있습니다.

`TSver` 브랜치에서는 TypeScript 버전의 코드를 확인할 수 있습니다. 

나중에 코드 확인 할때 사용해 보시는걸 권장합니다.
